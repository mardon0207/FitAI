# Последствия — фото-слоты

Сгенерируй комиксные/иллюстративные картинки и сохрани сюда **ровно с этими именами** (PNG или JPG — PNG предпочтительнее для прозрачности).

Стиль: мягкий комикс, чуть стилизованный, без медицинского хоррора — нужно **информировать, а не пугать**. Минимум текста — суть читается с картинки.

Рекомендация: квадратный формат, 512×512 px (потом масштабируется).

## Куда класть готовые файлы

Просто положи PNG в эту папку (`apps/web/public/consequences/`). Путь в коде: `/consequences/<slug>.png`.

Если файла нет — отображается emoji-плейсхолдер. Безопасно деплоить поэтапно.

## Как узнать что нужно сгенерировать?

Открой `apps/web/src/data/consequences.ts` — у каждой записи есть поле `photoSlug`. Имя файла = `<photoSlug>.png`.
